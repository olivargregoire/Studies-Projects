##  Where is what?

The repo grew a bit organically as I learned stuff, so here is a quick map to
find your way around (I basically write things down to actually understand them):

- **`Identity Security/`** — my deep dive into everything auth & identity. HTB
  modules I went through (Attacking Authentication Mechanisms, Broken
  Authentication), plus my own notes on OIDC, SAML 2.0 and Kubernetes RBAC.
  Probably the folder I spent the most time on.
- **`Concepts/`** — smaller standalone notes on tech concepts I wanted to nail
  down: TLS 1.3 (the full handshake, step by step), and GitHub Fine-grained PATs
  vs GitHub Apps. Has its own README with more detail.
- **`AI/`** — my machine learning / AI experiments: a facial recognition deep
  learning project, heart disease detection with ML, and CodeCrafter (my attempt
  at building my own Claude-Code-like tool).
- **`CPTS/`** — notes and service enumeration stuff from my CPTS pentesting prep.
- **`Mini Local App Architecure/`** — a small hands-on multi-container setup
  (nginx + two apps + a db) to play with local app architecture.
- **`Create and Start a Container Manually With runc/`** — a write-up where I
  build and start a container by hand with runc, to see what Docker actually does
  under the hood.

> Still a work in progress — I add to it whenever I learn something new. Feel
